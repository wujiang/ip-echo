* ip-echo

Return the remote request IP like ipecho.net/plain

** How to run it locally
1. src_sh{virtualven venv-ip}
2. src_sh{python setup.py install}
3. src_sh{pserve ip.ini}

** Hosting via uwsgi + nginx
